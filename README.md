# Mini Canvas Editor

React와 TypeScript를 활용해 구현한 간단한 캔버스 편집기입니다.

## 주요 기능

- 사각형 추가
- 도형 선택
- 드래그 이동
- 색상 변경
- 선택 도형 삭제
- 캔버스 영역 밖 이동 제한

## 사용 기술

- React
- TypeScript
- HTML
- CSS
- Vite

## 학습한 점

- React state를 활용한 도형 목록 관리
- 마우스 이벤트 기반 드래그 이동 구현
- 좌표 계산을 통한 UI 상호작용 처리
- 선택 상태 관리
- CSS position 기반 화면 배치
- TypeScript 타입 정의

## 실행 방법

```bash
npm install
npm run dev
